# 여민군(與民軍) 프로토타입 웹앱

AI 기반 민·관·군 협업 의사결정 플랫폼의 작동 방식을 보여주는 발표용 React 프로토타입입니다. 요청, 자원, 문서 생성은 로컬 데이터로 동작하며, 현장 지도는 정적 배포에서도 열리는 지도 embed를 사용합니다.

`여민군(與民軍)`은 시민과 함께한다는 뜻의 `여민(與民)`과 군을 결합한 명칭으로, 시민을 중심으로 행정기관과 군이 함께 지역문제를 해결하고 공공가치를 실현한다는 의미를 담고 있습니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 로컬 주소로 접속하면 됩니다. 기본 주소는 보통 `http://localhost:5173`입니다.

## 빌드 확인

```bash
npm run build
```

## 정적 사이트 배포 준비

- Vite `base`를 상대 경로(`./`)로 설정해 GitHub Pages 같은 정적 호스팅에서 경로가 깨지지 않도록 준비했습니다.
- `.github/workflows/deploy.yml`이 포함되어 있어 GitHub 저장소의 Pages 소스를 `GitHub Actions`로 설정하면 `main` 브랜치 push 시 자동으로 빌드·배포됩니다.

## 구현 범위

- 세종특별자치시 중심의 협업 대시보드
- 지도 embed 기반 민·관·군 지원 상황지도와 카테고리별 마커
- 마커 및 최근 요청 선택에 따른 AI 분석 패널 갱신
- AI 추천 자원 카드와 지원 요청 상태 변경
- 민·관·군 협업 프로세스 단계 선택
- 협조공문, 지원계획서, 결과보고서 자동 생성 모달
- 반응형 Tailwind CSS 레이아웃
